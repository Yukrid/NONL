# NONL
## To install
```
git clone https://github.com/Yukrid/Comlar.git
cd Comlar
mkdir build
cd build
cmake ..
make
sudo make install
cd ../..

mkdir build
cd build
cmake ..
make
sudo make install
```

## How to use
see help
```
nonl -h
```

```
nonl 
>>> AAA
BBB

END
```
